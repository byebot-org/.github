## 📖 About

byebot shields your server from bot attacks - mass join floods, ping floods, VPN abuse, and other automated threats. It works on both premium and cracked servers, requires no payment, and gives server owners full control through a modern web dashboard.

## ✨ Features

- 🛡️ **Comprehensive Bot Protection** - Protects against mass joins, ping floods, and VPN abuse
- 🎮 **Universal Compatibility** - Works on both premium and cracked Minecraft servers
- 💰 **100% Free** - No hidden costs or premium tiers
- 🎛️ **Modern Dashboard** - Full control through an intuitive web interface
- ⚙️ **Highly Configurable** - Customize protection rules to fit your server needs

## 🤝 Contributing

We welcome contributions! To contribute code to this project, please follow these guidelines:

- **Organized and focused PRs**: Each pull request should have a specific and clear objective. Avoid covering multiple features or changes in a single PR.
- **Clean code**: Keep your code readable, well-structured, and following the project's standards.
- **Easy review**: By keeping PRs small and focused, you make the review process more efficient and faster.

### Code Quality Standards

- Follow the existing code style
- Write clear commit messages
- Test your changes thoroughly
- Update documentation as needed
